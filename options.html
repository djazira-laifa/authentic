Authentic settings

Sightengine credentials

Authentic uses the Sightengine API to classify images. You need your own API user and API secret. Sign up free at dashboard.sightengine.com, then copy both values from the API keys section of your dashboard. Both are stored only in your browser and sent directly to Sightengine, never to anyone else.

Detection threshold

How confident the classifier must be before a pin is flagged as AI. Lower values catch more content but may flag some real photos as AI. The default of 0.5 is balanced for Sightengine's scores.

0.50

Daily check limit

Maximum number of new images checked per day. Each check costs 5 Sightengine operations, and the free tier includes about 2000 operations per month (roughly 400 checks). The default of 100 per day keeps you safe. Cached pins are always free and don't count.

100

Reset

Clear the local cache of classified pins, or reset your block count if you want to start fresh.
