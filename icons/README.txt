Icons needed before publishing:

* icon16.png  (16 x 16 px)   used in the toolbar and the extensions list
* icon48.png  (48 x 48 px)   used in the extensions management page
* icon128.png (128 x 128 px) used in the Chrome Web Store listing

Design suggestion: a simple monochrome mark on a transparent or off-white
background. The brand is "Authentic", so something minimal and confident,
not playful. Avoid AI-generated icons (it would be a bit ironic).

Tools you can use:
* Figma (free) to design the master file at 128x128, then export at all sizes.
* Or generate a simple letter mark with https://icon.kitchen/

Place the three PNGs in this folder, then they will be picked up
automatically by manifest.json.
